# TaskFlow Frontend

React frontend for TaskFlow – Secure Personal Task Manager.

## Features

- User Login
- User Register
- Dashboard
- Add Tasks
- Delete Tasks
- Logout
- Responsive UI

## Tech Stack

- React.js
- Axios
- React Router DOM

## Setup Instructions

1. Clone repository
2. Run npm install
3. Run npm start

## Runs On

http://localhost:3000
